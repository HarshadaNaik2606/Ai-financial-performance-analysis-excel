# 📊 AI-Driven Financial Performance Analysis: NVIDIA vs Intel  
## Excel-Based Analytics, Modelling & Dashboarding  

---

## 🚀 Project Overview  

This project delivers a data-driven comparative analysis of NVIDIA and Intel, evaluating how strategic adaptation to technological change impacts financial performance.

The entire workflow—from data collection and cleaning to analysis, modelling, and visualisation—was executed using Microsoft Excel, demonstrating the ability to handle large, unstructured datasets and transform them into actionable business insights.

---

## 🧩 Business Context  

Historically, Intel dominated the semiconductor industry, powering the majority of personal computing systems.

In contrast, NVIDIA initially focused on graphics processing for gaming.

With the rapid shift towards artificial intelligence (AI), cloud computing, and data centre infrastructure, NVIDIA strategically repositioned itself and capitalised on emerging technologies.

Intel, however, demonstrated slower adaptation.

This project evaluates how these strategic differences are reflected in financial performance over time.

## 📂 Data & Methodology  

### Data Source  
- Public financial data (Macrotrends)  
- Quarterly metrics:
  - Net Income  
  - Gross Profit  

### Data Processing (Excel-Based)  
- Cleaned and structured raw datasets  
- Standardised inconsistent formats  
- Converted text-based financial values into numeric format  
- Prepared datasets for analytical modelling  

---

## 🛠️ Excel Techniques & Skills Demonstrated  

This project was completed entirely in Excel, showcasing end-to-end analytical capability:

### 🔹 Data Preparation  
- Cleaning and structuring unorganised datasets  
- Handling missing and inconsistent values  

### 🔹 Data Transformation  
- Pivot Tables  
  - Converted quarterly data into yearly summaries  
  - Aggregated Net Income and Gross Profit  

### 🔹 Analytical Techniques  
- Year-on-Year (YoY) Growth Analysis  
- Compound Annual Growth Rate (CAGR)  
- Cumulative Growth Index  
- Rolling 3-Year Average  

### 🔹 Excel Functions & Formulas  
- `XLOOKUP` → advanced data retrieval and replacement for VLOOKUP  
- `VLOOKUP` → structured table lookups  
- `IF` → conditional logic  
- `IFERROR` → error handling  
- `AVERAGE` → rolling averages  
- `ABS` → magnitude of change analysis  
- Custom-built formulas for:
  - YoY growth  
  - CAGR  
  - Growth index calculations  

### 🔹 Data Visualisation  
- Pivot Charts  
- Trend analysis (time-series comparisons)  
- Interactive dashboard development  

---

## 📈 Key Insights  

- NVIDIA demonstrates strong and consistent growth in both net income and gross profit  
- Intel exhibits higher volatility and comparatively weaker performance trends  
- Growth metrics (YoY and CAGR) highlight a widening performance gap between the two companies  
- Long-term trends confirm NVIDIA’s sustained growth trajectory versus Intel’s inconsistency  

---

## 💡 Strategic Takeaways  

- Adaptability to AI-driven technological shifts is a critical driver of long-term business success  
- Early investment in emerging technologies creates sustained competitive advantage  
- Financial performance directly reflects strategic decision-making and market positioning
